# General Purpose Web
> 아래 세가지 기준을 중심으로 중장기 웹 기술을 연구한다.
- Edge Performance
- Cross Platform
- Common Platform

## Naive Technical Stack
- Client:
    - React (with WebAssembly)
    - Canvas (with WebGL)
- Server:
    - Express (with C/C++ based Engine)
- WebRTC
